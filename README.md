# 🚀 LandLink - Student Room Finder

LandLink is a smart platform that helps students find affordable and verified rooms near their college.

---

## 🌟 Features

- 🏠 Post Rooms with Image/Video
- 🔍 Search Rooms by City & Budget
- 👤 User Authentication (Login / Signup)
- 📸 Upload Room Photos
- ⚡ Clean & Modern UI
- 📱 Responsive Design

---

## 🛠️ Tech Stack

- Frontend: HTML, CSS
- Backend: PHP
- Database: MySQL
- Server: XAMPP
